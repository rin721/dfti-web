# Project Brief: DFTI (Delta Force Type Indicator)

## 1. Project Goal
Build a personality archetype testing platform for the *Delta Force* universe, mapping users to character/faction archetypes based on MBTI frameworks and community meme culture.

## 2. Product Positioning
An entertainment-oriented fan project that bridges the gap between official game lore and community-driven interpretations (memes, fan-fic impressions).

## 3. Core Boundaries
- **Character Pool**: GTI, Asara, Haverk, and Battlefield Archetypes.
- **Data Layers**: Official (Canon), Community (Fanon), Mixed.
- **Atmosphere**: Tactical, Dark, Immersive, Shareable.

## 4. Naming Conventions
- Project Name: DFTI (Delta Force Type Indicator)
- Chinese Subtitle: 《三角洲行动》角色人格测试站

## 5. Technical Stack
- **Frontend**: React 18 (Vite)
- **Styling**: Tailwind CSS 4.x
- **Animation**: Motion (motion/react)
- **Icons**: Lucide React
- **Data**: Local JSON modules

## 6. Key Constraints
- Mobile-first responsive design.
- Clear distinction between official lore and fan interpretations.
- Result page must be visually dense and "screenshot-worthy".
